# Luo Browser

## Overview
The Luo Browser is a dual-access web browser designed to support both human users and AI agents. It utilizes the `luo://` protocol to provide a unique browsing experience tailored for both user types.

## Features
- Dual-access for humans and AI agents.
- Support for the `luo://` protocol.
- User-friendly interface for browsing.
- AI agent optimization for web interactions.
